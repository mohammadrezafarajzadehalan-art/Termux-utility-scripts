ویژگی‌های اصلی این اسکریپت:

1. تشخیص سرطان‌های اجتماعی

· مادی‌گرایی (مرحله پیشرفته)
· فردگرایی (مرحله حاد)
· سطحی‌نگری (مرحله پیشرفته)
· بی‌اعتمادی (مرحله متوسط)
· فقدان همدلی (مرحله حاد)

2. نظام طبقاتی جدید

· ۵ طبقه اجتماعی بر اساس ترکیب ارزش انسانی/مالی
· هر طبقه مثال‌های واقعی از محله

3. نقشه کامل محله

· ۷ شخصیت با مشخصات دقیق
· برچسب‌های اجتماعی خودکار
· ارزش‌گذاری طبقاتی

4. معیارهای وارونه

· ۷ ارزش انسانی که بی‌ارزش شده‌اند
· ۷ ارزش مادی که تبدیل به معیار اصلی شده‌اند

5. مکانیزم‌های دفاعی جامعه بیمار

· ۶ مکانیزم روانی برای توجیه وضعیت موجود
· مثال‌های واقعی از هر مکانیزم

6. شبیه‌سازی‌های تعاملی

· محاکمه اجتماعی پسران
· چرخه معیوب فقر
· دیالوگ‌های روزمره محله
· تحلیل روانشناختی عمیق

7. راهکارهای درمان سه‌سطحی

· درمان‌های فوری
· اصلاحات ساختاری
· تغییرات فرهنگی

8. گزارش حرفه‌ای

· خروجی Markdown با ساختار استاندارد
· قابل استفاده برای پژوهش‌های اجتماعی
· مستندسازی کامل وضعیت

پیام اصلی:

این اسکریپت نشان می‌دهد که:

1. پسران همسایه قربانیان سیستم هستند - نه مقصران
2. جامعه بیمار، انسان‌های سالم را بیمار می‌کند
3. چرخه معیوب، شکست را به ارث می‌گذارد
4. شفا ممکن است - اما نیاز به جراحی فرهنگی دارد
